# React Dynamic Layout

React Dynamic Layout is a dock layout system inspired by [Golden Layout](https://golden-layout.com), made with react.js

## Features

* Easy way to create layouts
* Multiple levels of hierarchy
* Float windows
* Panel stack
* Multiple themes
* Resizable
